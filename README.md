# Home Services Logbook

Home Services Logbook is a simple self-hosted web app for keeping a household service history in one place. The basic idea is to be the digital version of the spiral-bound notebook next to the phone, or the filing cabinet for all the paperwork.


## Implemented Features
- Vendor Management & Timeline
- Attachments: add, remove, download
- Labels
- Calendar export: generate downloadable `.ics` files.

#### Simple Lightweight Stack
- Python
- FastAPI
- Jinja2 templates
- SQLite
- Local filesystem uploads
